# Financial Research Agent

An AI-powered financial research assistant built using:

- OpenAI GPT-4o
- Yahoo Finance
- Tavily Search
- Python

## Features

- Stock analysis
- News analysis
- Risk assessment
- Investment recommendations

## Run

```bash
pip install -r requirements.txt

python main.py
```

## Example

```bash
Enter Stock Symbol: NVDA
```

Generates:

- Company Overview
- Bull Case
- Bear Case
- Risks
- Recommendation
